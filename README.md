# VendmoTech_Security_Audit_Report.
A full cybersecurity audit &amp; risk assessment for Vendmo Tech, a fictional fintech. Covers 14 vulnerabilities (Critical–Low), a 5×5 risk matrix, NIST CSF &amp; ISO 27001 framework mapping, and 12 prioritised remediation recommendations. Tools: OWASP ZAP, Nmap, Wireshark. Aligned to POPIA &amp; PCI-DSS.
